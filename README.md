# Django Job Portal

A modern Job Portal web application built using Django with role-based authentication, profile management, job posting, and application tracking features.

---

# Tech Stack

## Backend
- Python
- Django

## Frontend
- HTML
- CSS
- Bootstrap 5
- JavaScript

## Database
- SQLite

---

# Project Features

## Authentication System
- User Registration
- User Login
- User Logout

## Role-Based Authentication
- Candidate Role
- Recruiter Role

## Candidate Features
- Profile Image Upload
- Resume Upload
- Skills Section
- Education Section
- Experience Section
- Bio Section

## Recruiter Features
- Company Name
- Company Website
- Company Description
